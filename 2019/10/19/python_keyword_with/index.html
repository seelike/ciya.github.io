with:

  1. 基本语法格式： with exp [as target];

target 可以是变量或者元组，存储exp返回的结果；

  1. With是一种上下文管理协议，其表达式其实是try - finally 的简写形式，但没有except 功能；
  2. 使用with处理的对象必须包含方法 enter()和exit()；
  3. enter()在方法体执行前运行，初始化代码块，并将返回值赋值给as关键字后的变量;
  4. exit() 在方法体执行结束后运行，关闭代码块，释放资源，如文件使用后关闭等。即便代码块抛出异常也调用exit() 退出代码块。
