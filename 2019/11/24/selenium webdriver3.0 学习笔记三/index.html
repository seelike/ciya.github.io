  1. 在python中安装WebDriver：

完成python，pip工具的安装后，在cmd下输入“pip install selenium”，

完成安装后输入”python” 进入python 交互模式， 输入“import selenium” 没有报错即成功安装selenium。

  1. 安装浏览器驱动：

Firefox 浏览器的驱动： https://github.com/mozilla/geckodriver/releases

Google浏览器驱动： http://npm.taobao.org/mirrors/chromedriver/

选择适合自己操作系统的版本进行下载。
